# project_brief

## What
voidrun — 2-4 player online co-op roguelike. Top-down twin-stick. Procedural sector layouts, run length 30-45 min, permadeath per run with persistent meta unlocks.

## Pillars
1. **Tight feel.** 60Hz sim, sub-100ms perceived input latency on LAN, <150ms on transatlantic.
2. **Build variety.** Weapons + mods + relics combine. Target ~10^4 viable build seeds at launch.
3. **Co-op that matters.** Revives, shared loot pool with claim timer, friendly fire off but knockback on.

## Scope (v1.0)
- 4 biomes, 8 boss encounters, ~40 weapons, ~120 relics, ~30 enemy archetypes.
- Steam launch (Win x64). Deck-verified targeted post-launch.
- No PvP. No mod support v1.

## Out of scope
- Cross-play. Mobile. Dedicated servers (peer-host only at v1).
- Seasonal content pipeline (post-launch concern).
