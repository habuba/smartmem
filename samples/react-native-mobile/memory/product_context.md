# Product context

## Who uses this
- Day hikers and weekend backpackers, ages 25-55, mixed iOS/Android.
- Mostly solo or small-group; not ultralight thru-hikers (different segment, AllTrails/Gaia own it).
- Comfortable downloading content before a trip; not comfortable configuring map layers manually.

## Core scenarios
1. **Night before**: at home on wifi, user imports a GPX from a friend, downloads the tile pack for that region, pins the route. Goes to bed.
2. **Trailhead**: in the parking lot, signal is one bar. Opens app — route is already there, map renders from cache, no spinner.
3. **On trail**: phone in pocket, screen off. Recording in background. Off-route triggers a single haptic, not a notification storm.
4. **After**: back in signal, track auto-uploads. User shares a link, friend opens it (web fallback if they don't have the app).

## Monetization (later)
- Free tier: 3 saved routes, 1 active tile pack.
- Pro ($4.99/mo or $39/yr): unlimited routes and packs, weather overlays, heart rate from Bluetooth.

## Key non-goals from product
- We do not show ads. Ever. It is part of the brand.
- We do not require an account to use the app. Sign-in is optional and only needed for cross-device sync and sharing.
